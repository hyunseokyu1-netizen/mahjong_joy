# 🀄 Mahjong Joy — 마작 조이

**짝 맞추기의 재미만 남긴 심플 마작.** 족보도 역도 없이, 패 14장을 "몸통 4개 + 머리 1개"로 만들면 승리입니다.

전통 마작을 낯설어하는 사람들이 진짜 어려워하는 건 규칙이 아니라 복잡한 점수 체계(역·판수·부수)라고 생각했습니다. 그래서 규칙은 조합 하나만 기억하면 되도록 걷어내고, 대신 승리 점수는 "기본 100점 + 더하기 보너스, 곱하기 보너스"로 재구성해 영수증처럼 한 줄씩 보여주는 연출을 넣었습니다.

## ✨ 특징

- **규칙 하나만 기억하면 끝** — 몸통(스트레이트/트리플) 4개 + 머리 1개. 족보 암기 없음.
- **점수 영수증 연출** — 완성하면 보너스 항목이 딩~ 소리와 함께 한 줄씩 나타나며 소계가 올라간다.
- **AI 대전** — 나는 항상 좌석 0, 나머지 3자리는 AI가 채워 바로 4인 대국.
- **LAN 멀티플레이** — 같은 Wi-Fi에서 방을 만들면 최대 4명이 함께. 서버 없이 호스트 폰이 유일한 심판 역할을 하고, 참가자는 행동만 전송한다. UDP 브로드캐스트로 방을 자동 발견하고, 순간 끊김은 자동 재접속으로 복구된다.
- **응답 제한시간** — 패 가져가기 프롬프트에 15초 타이머. 다른 사람이 고민 중이면 "🤔 고르는 중..." 배너로 알려준다.
- **3개 언어** — 한국어 · 中文 · English, 기기 언어 자동 감지.
- **초보자 모드** — 점수 계산 없이 짝 맞추기만 즐기는 스위치.
- **직접 합성한 사운드** — 외부 에셋 없이 파이썬으로 합성한 효과음 (`tool/gen_sfx.py`).

## 🛠 기술 스택

Flutter · Dart · `dart:io` 소켓(LAN 멀티플레이) · Provider · `audioplayers` · `shared_preferences`

## 프로젝트 구조

```
lib/
  models/       타일 등 데이터 모델
  logic/        게임 엔진, 점수 계산 (UI와 완전히 분리된 순수 Dart)
  ai/           AI 상대 전략
  net/          LAN 멀티플레이 프로토콜 · 호스트/클라이언트 세션 · 방 발견
  ui/           화면, 컨트롤러(TableController로 로컬/호스트/클라이언트 공용)
  sound/        효과음 재생
  i18n/         3개 언어 문자열
  settings/     언어·초보자 모드 저장
tool/
  gen_sfx.py    효과음 WAV 합성 스크립트
  gen_icon.py   앱 아이콘 생성 스크립트
test/           단위 테스트 + 실제 소켓을 쓰는 LAN 통합 테스트
```
